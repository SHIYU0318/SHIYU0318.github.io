自我介紹

大家好 我是士育 ShiYu

目前就讀 南大附中

高二 普通科 自然組 喜歡一切有關資訊領域的事物

目前正自學程式設計 主要使用的程式語言為 C++ 也會一點 Python

為了記錄我所學到的資訊相關知識 創建了此 Blog

學習歷程

國小資訊課有玩過 Scratch

那時還不知道這算是入門的程式設計

只是把它當成遊戲來玩玩 覺得很有趣

國中就讀 嘉義縣竹崎高中(國中部)

在國二時接觸到了程式設計競賽

那時使用到 App Inventor2

透過資訊老師的指導

開發出一個 Android App

後來開始學習 C 語言

使用 Arduino UNO 板

實做出 IoT 物聯網的設備

國三開始學習 Python

在比賽作品中運用到了 AI 神經網路辨識的功能

高一開始接觸競技程式

加入了各種 Discord 伺服器資訊討論交流社群獲取更多學習資源

也開始學習 C++ 並使用線上評測平台來解題目提升自己的程式能力

(在高一期間無意間激起了對量子力學的興趣)

此 Blog 也在 2022/12/01 正式建立

為了提升南大附中的資訊風氣

我與 YuDong 在 2023/06

共同創立了 南大附中資訊研究社第一屆

希望能營造一個良好開放的資訊社團

詳細創社計畫可參閱南大附中 資訊研究社 創社計畫

目前也正積極規劃社團運作相關事務

為了能對資訊社群做出一點貢獻

我在升高二時加入了 SCIST 南台灣學生資訊社群 的行政組

目前擔任 112 學年度 第四屆 小編

負責經營粉絲專頁 社群管理的工作

高二時也參加了 陽明交通大學 人才培育計畫 人工智慧與深度學習

學習關於 AI 的知識

希望透過此 Blog 紀錄我的學習過程 分享學習資源

我也將持續學習 不斷努力 提升自己的資訊能力
